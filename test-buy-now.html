ðŸ“‹ How It Works

  1. User clicks "Buy Now" on product detail page
  2. Product data is collected (name, price, size, quantity, image)
  3. Data is encoded and added to URL parameters
  4. User is redirected to checkout page
  5. Checkout page reads the URL parameters
  6. Order summary is displayed with product details
  7. User can edit order or add more products
